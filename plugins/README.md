# Plugins

This directory contains plugin configurations for Claude Code.

## Usage

First, add the plugin:

`/plugin marketplace add XXX`

Next, install it:

`/plugin install XXX`

Finally, configure it if applicable.

## List

[HUD](https://github.com/jarrodwatts/claude-hud)

- `/plugin marketplace add jarrodwatts/claude-hud`
- `/plugin install jarrodwatts/claude-hud`
- `/claude-hud:setup`

[Claude Mem](https://github.com/thedotmack/claude-mem)

- `/plugin marketplace add thedotmack/claude-mem`
- `/plugin install thedotmack/claude-mem`

[Claude Frontend](https://github.com/anthropic/claude-code)

- `/plugin marketplace add anthropics/claude-code`
- `/plugin install frontend-design@claude-code-plugins`
